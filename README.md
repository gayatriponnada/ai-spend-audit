# AI Spend Audit

A free web app that helps startup founders and engineering managers
find out if they are overspending on AI tools like ChatGPT, Cursor,
Claude, and GitHub Copilot — and shows exactly how much they can save.

## Decisions
1. Chose React + Vite for fast setup and my existing experience
2. Chose Supabase for backend — free tier, easy setup, real database
3. Chose Vercel for deployment — free, connects directly to GitHub
4. Chose Tailwind CSS for styling — fast to build, responsive by default
5. Chose Resend for emails — simple API, generous free tier
